This Website
The Point of this website?

Having learn many technologies outside my works and inside it. It is hard to keep track of them, so I building a website to host some of what I done so far. The central point is these Skill Trees. and this page. .These trees listed the coding languages and skillsets that I've used before, and haven't forget that I used them.

Why so many external links?

Github is the website where I stored most of my coding projects. This website is made to display the results of the codes, which Github can't display as well, though its ability for Markdown file made for longer readings in my experience.

Most of my previous Data Visualization is displayed as stand-alone HTML files. This site is new, I don't intend to incorporate all other web into it.

Most Websites I've seen has more features?

It would have been much, more faster and visually appealing if I bought a domain from "Wordpress" or "Squarespace", or a template from a full-time web designer.

I built this mostly from scratch (with help from Open Source projects, of course). Easier and less complicated ways to make a website existed. But this is an opportunity to learn web design frameworks and tools such CSS, and Bootstraps, and it is FREE.

Why Three Pages? Too Many Clicks? Too Much Text?

This is not a website if I want to apply to a UI/UX position. I personally prefer simple sites with text, image and limited Interactivity.

I do know of the existence of Javascript Frameworks to make this more faster. But I'm not planning on learning it today, at least until my current project(s) is done. Not sure how fast all images in one page will load.

This website is designed to be simple as I can. All these texts might soon be tooltips, off-canvas, or collapsible cards. But right now, I really want to go back to Data-related projects.

Missing Links? Terrible spellings and grammars?

That' s why most people I know (except a few experienced web designers) generally don' t build a website for personal use and rather pay for it.

Its tiring to check all the grammar and spellings by myself, I rather do other works for now. After a week, with a clearer head, I would fix them.
